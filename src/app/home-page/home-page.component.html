Título del ejemplo

Este es un ejemplo de cómo colocar una imagen a la derecha de un texto utilizando HTML y CSS. El contenedor utiliza el modo flexbox para distribuir el espacio de manera eficiente. La propiedad "flex" se aplica al elemento "texto" para que ocupe todo el espacio disponible que no sea utilizado por la imagen.
